netman-app
==========

Android App NetMan for IT Specialist

Download the official Eclipse Android SDK under
http://developer.android.com/sdk

1. Open Eclipse.
2. Open Help > Install New Software.
3. Choose the "--All Available Sites--" and search EGit & the EGit Github plugin to install.
4. Open the Git Repository Explorer.
5. Clone the Project.
6. Remove it from Eclipse (don´t delete it from harddisk).
7. Rename the folder "netman-app" to "netman" in your workspace.
8. Import your project again in Git(over local path).
9. Right click on your git project and say import.
10. Choose import "existing projects" and open the project folder in your workspace.

Now you can work with the app ;)
